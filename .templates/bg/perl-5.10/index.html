Здрасти! Аз съм твоето {{name}} приложение което създаде на STARTAPP.bg

Качи на сървъра промените които си направил

StartApp използва Git SCM за кода на твоето приложение и Secure Shell (SSH) protocol за да можеш лесно да го достъпваш. За да качваш или сваляш код от клауда на StartApp ти трябва да имаш публичен SSH ключ.
Ако имаш вече такъв, можеш да го добавиш през Уеб Конзолата или да инсталираш StartApp APP command line tool.

Работа с твоето локално Git хранилище

Ако създадеш своето приложение през командния ред и ъплоуднеш своя SSH публичен ключ, app автоматично ще свали копие на хранилището на твоето приложение от сървъра (на жаргона на Git това се нарича клониране) на твоя компютър.

Ако създадеш своето приложение през Уеб Конзолата, ще трябва ръчно да клонираш хранилището от сървъра на тоя компютър. Копоирай Git URL-а който ти се изписва след като приложението ти е създадено и стартирай следната команда: 

\$ git clone <git_url>

# Commit-ни промерните които си направил
\$ git commit -am "Some commit message"

# push-ни към StartApp
\$ git push

Управление на твоето приложение

Уеб Конзола - Можеш да използваш Уеб Конзолата на StartApp за да обогатяваш приложението си с допълнителни технологии като MySQL, MognoDB, Redis, Memcache и други. Да управяваш хората с които работиш, да регламентираш достъпа който те тябва да имат, да добавяш custom домейн имена от вида www.example.com и тнт.

Command Line Tools - Инсталирайки StartApp APP command line tool за Windows, MacOS или Linux ще имаш пълен контрол на твоята cloud среда. Намери още информация как да управляваш твоите приложения в Документацията.
